— Quick Start —

📎 第一次使用

  1. 點上方「📎 連結 CSV」按鈕
  2. 在彈出視窗選擇本資料夾的 books.csv
  3. 瀏覽器詢問權限時點「儲存變更」授權寫入
連結後按鈕變綠色「✓ 已連結」。
之後 拖移、編輯、刪除 都會即時寫回 books.csv 檔案，無需手動匯出。

📖 閱讀

  • 點書本：書本浮起翻面，看詳細資料
  • 拖移書本：變更陳列順序（自動存檔）
  • 按 Esc 或點背景闔上

✎ 編輯

  • 點「✎ 編輯模式」切換
  • 點書本：修改書名、作者、顏色等
  • 點紅色 ×：刪除書本
  • 「＋ 新增書本」加入新書

⚠ 不支援自動寫檔時

  • Safari / Firefox 無法直接寫檔
  • 請用「⬆ 載入 CSV」手動載入
  • 變更後「⬇ 匯出 CSV」覆蓋原檔
建議使用 Chrome / Edge / Brave / Arc 等 Chromium 瀏覽器以啟用完整功能。
